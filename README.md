Version la plus avancé dans DeplacementJoueur_Panzera > index.html
## Créateur : 
* SOLANA Louis

* PANZERA Alexis

### Création d'un jeu de Survie qui consiste à rester le plus longtemps en vie contre une invasion bacterienne.

### Module permettant son developpement :
* Module lancement d'une partie. 
* Module éléments : Création des éléments (armes, personnage,bacterie)
* Module bacteries : Apparition et destruction des bacteries.
* Module son : Diffusion d'un son lors de la destruction d'une bactérie.
* Module déplacement : Gestion du déplacement du personnage et de la balle tirée. 
* Module score : Comptabilisation du score.
* Module fin de partie.
