# Flipper-Game

## Créateur : 
* SOLANA Louis
* BAROUKH Yoni

### Création d'un jeu de Flipper en Javascript.

### Module permettant son developpement :
* Module lancement d'une partie. 
* Module éléments : Création des éléments (bille, bumpers, kickers, rampes...)
* Module palette : Gestion des évènements clavier pour manipuler les batteurs.
* Module son : Diffusion d'un son selon l'impact entre la bille et les éléments.
* Module déplacement : Gestion du déplacement de la bille selon les collisions et le renvoie de la bille par les batteurs.  
* Module score : Comptabilisation du score.
* Module fin de partie.
